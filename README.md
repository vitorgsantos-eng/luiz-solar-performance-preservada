# Luiz Solar — Performance Preservada

Landing page institucional da Luiz Solar voltada para manutenção, diagnóstico, limpeza técnica e reparos em sistemas fotovoltaicos.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Execução local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
